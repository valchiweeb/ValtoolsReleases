# ValtoolsReleases
Download the latest release from the Releases page.

### New Features
- **Game Injection**: Integrate games to Steam
  - Enter Game ID or paste Steam URL
  - Automatic DLC detection
  - One-click Activate Inject
  - Restart Steam from app
  - Integrate With Online-fix.me ( Coming Soon )

### How to Use
1. Open Game Injection menu
2. Click 'Activate Inject'
3. Enter Game ID
4. Click 'Add Game'
5. Restart Steam
